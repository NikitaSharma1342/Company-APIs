# Company-APIs
API to access the information of companies 

# Description 
This application is created with the help of RESTful concepts, sqlalchemy database concepts, APIs, Postman and Flask.

# Frameworks and Libraries used
* Flask
  * render tempelate
  * jsonify
  * request
* flask-sqlalchemy
